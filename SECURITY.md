# Security Policy

## Scope

This repository contains a public demo site, project documentation, and token metadata drafts for CNYAN / ChallengeCAT.

## Reporting

Do not publish exploit details in public issues. If a private reporting channel is not available, open a minimal GitHub issue that describes the affected area without sensitive details.

## Identity Safety

Do not submit or request:

- personal names
- private addresses
- private emails
- personal wallet ownership claims
- operator identity information
- location or doxxing material

All security discussion should remain project-focused.
