## Solana Concepts Used

- **Mint account**: the canonical address identifying a token on Solana.
- **Token account**: an account holding token balance for a wallet and mint.
- **Associated Token Account**: a token account derived from wallet and mint.
- **Metaplex Token Metadata**: metadata account linked to a mint for name, symbol, URI, and visual metadata.
- **Off-chain metadata JSON**: a JSON file used by wallets and explorers for richer display information.
