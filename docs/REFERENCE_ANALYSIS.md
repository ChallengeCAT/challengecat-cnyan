# Reference Analysis: quest-mvp

Reference reviewed:

```text
https://github.com/ahhimquesting/quest-mvp
```

## What Was Learned

The reference repository presents a social challenge protocol with:

- challenge posting
- funded rewards
- claim flow
- video proof
- AI verification
- escrow settlement
- API / web / contract separation
- anti-abuse rules
- product PRD and design overview

## What Was Not Copied

This repository does not copy:

- source code
- UI components
- exact copy
- branding
- token address
- product name
- images
- repository structure verbatim

## What Was Adapted

The abstract protocol pattern was adapted into original CNYAN concepts:

- `Post -> Claim -> Prove -> Verify -> Settle`
- CNYAN posting fees
- CNYAN/SOL reward settlement
- Quest Protocol documentation
- DEX settlement and liquidity thesis
- AI verification as one possible trust layer
- privacy-safe, project-first language
