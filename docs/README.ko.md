# CNYAN 한국어 개요

CNYAN은 ChallengeCAT의 커뮤니티 미션, Proof, 토큰 보상 콘셉트 프로젝트입니다.

## Token

- CA / Mint: `Ayfr5kqa2VsXzo8D2CB59pUaLCeG5GnVHod4ixutpump`
- Network: Solana
- Symbol: CNYAN

## 저장소 목적

이 저장소는 다음 내용을 보여주는 데모 사이트와 프로젝트 문서입니다.

- 챌린지 생성
- Proof 제출
- 바운티 맵
- 경로 정보 맥락
- AI 답변 UI
- CNYAN/SOL 보상 경험

## 익명성 원칙

문서는 프로젝트 중심으로 작성합니다. 개인 이름, 개인 연락처, 위치 정보, 개인 지갑 소유 주장은 포함하지 않습니다.
